# IPL-Analysis-Dashboard
IPL Analysis Dashboard is a data analytics and visualization project built on Indian Premier League data sourced from Kaggle. It involves data cleaning, transformation, SQL analysis, and interactive visualization using Power BI to deliver actionable insights on team, player, and match performance. 🏏📊

The dataset was collected from Kaggle, using publicly available IPL match and player data.

Data was gathered in structured formats such as CSV and Excel files.

Raw data cleaning was performed to handle missing values, duplicates, and incorrect records.

Data normalization and transformation were applied for consistency.

Excel Power Query was used for data cleaning and preprocessing.

Data modeling was created to connect multiple datasets (matches, teams, players, venues).

SQL was used for querying, filtering, and aggregating large datasets.

Feature engineering was applied to create new performance metrics.

Key KPIs such as total runs, wickets, strike rate, economy rate, and win percentage were derived.

Data visualization was performed using Power BI.

Interactive dashboards were designed with slicers and filters.

Team-wise and player-wise performance analysis was included.

Season-wise and match-wise trend analysis was implemented.

Venue-wise performance and pitch behavior analysis were visualized.

Head-to-head team comparison dashboards were created.

The project supports strategic decision-making for team selection and match planning.

It helps in performance evaluation and talent identification.

The dashboard improves sports analytics and match insights.
